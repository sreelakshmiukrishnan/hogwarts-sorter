# 🧙‍♂️ Hogwarts House Sorter

A magical Harry Potter House Sorting web app built with React!

## ✨ Features
- 🏰 Animated magical landing page
- 🎩 Custom animated Sorting Hat
- 📜 8 personality quiz questions  
- 🏆 House result reveal with confetti
- 🎵 Background ambient music
- ✨ Floating sparkle cursor
- 📱 Fully mobile responsive

## 🛠️ Built With
- ⚛️ React + Vite
- 🎬 Framer Motion
- ⭐ tsParticles
- 🎨 React Icons
- 🎊 React Confetti
